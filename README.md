# ✈️ AeroNavigator | Skypro Python Developer
## Курсовая работа: Мониторинг воздушных судов

Проект представляет собой сервис мониторинга воздушных судов с использованием открытых API (Nominatim, OpenSky).

---

## 🎯 Реализованный функционал

### 📦 Класс `Aeroplane` (Воздушное судно)
Модель самолёта с валидацией данных.

**Атрибуты:**
- `icao24` (str) — уникальный идентификатор транспондера
- `callsign` (str) — позывной (по умолчанию "N/A")
- `country` (str) — страна регистрации
- `altitude` (float) — высота полёта (м)
- `velocity` (float) — скорость полёта (м/с)

**Методы:**
- `cast_to_object_list(data)` — преобразование данных OpenSky в список объектов
- Валидация типов и значений

### 🌍 Класс `AeroplanesAPI` (Работа с API)
Интеграция с внешними сервисами.

**Возможности:**
- Получение координат стран через Nominatim
- Запрос данных о самолётах через OpenSky API
- Обработка ошибок подключения и ответов
- Формирование bounding box по координатам страны

### 💾 Класс `JSONStorage` (Хранилище)
Сохранение данных в JSON-файл.

**Возможности:**
- Добавление данных без дубликатов
- Загрузка из файла
- Автоматическое создание файла при отсутствии
- Выборка данных по критериям (`get`)
- Удаление данных по критериям (`delete`)

### 🖥️ CLI-интерфейс (`main.py`)
Интерактивная работа с пользователем через консоль.

**Доступные действия:**
- Ввод страны для мониторинга
- Топ N самолётов по высоте
- Фильтр по стране регистрации (выбор из списка реальных стран в данных)
- Фильтр по диапазону высот
- Пагинация при большом количестве результатов
