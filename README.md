# AeroNavigator ✈️

Мониторинг воздушных судов через открытые API (Nominatim, OpenSky).  
Получает реальные данные о самолётах, сохраняет в JSON, позволяет фильтровать и выводить топ N по высоте.

---

## 🚀 Возможности

- Получение координат стран через Nominatim
- Запрос данных о самолётах через OpenSky API
- Фильтрация по стране регистрации и диапазону высот
- Топ N самолётов по высоте
- Сохранение данных в JSON-файл (без дубликатов)

---

## 🛠️ Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Viktor84-code/AeroNavigator.git
cd AeroNavigator

# 2. Установить зависимости
poetry install

# 3. Запустить программу
poetry run python main.py
```
## Что сделано

- ✅ Абстрактный класс для работы с API (`BaseAPI`)
- ✅ Класс `AeroplanesAPI` для интеграции с Nominatim (координаты стран) и OpenSky (данные о самолётах)
- ✅ Модель `Aeroplane` с валидацией и преобразованием данных
- ✅ Класс `JSONStorage` для сохранения данных в JSON-файл
- ✅ CLI-интерфейс с фильтрацией по стране, высоте и топ N
- ✅ Обработка ошибок API и пользовательского ввода
