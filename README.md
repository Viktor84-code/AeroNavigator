# AeroNavigator ✈️🧭

Промышленная система мониторинга и анализа авиационного трафика. 
Проект разрабатывается с соблюдением принципов SOLID, архитектурных стандартов и высокого покрытия тестами.

## 🚀 Основные возможности
- **Сбор данных**: Интеграция с внешними авиа-радарами (API).
- **Валидация**: Автоматическая очистка "грязных" данных и фильтрация ошибок.
- **Масштабируемость**: Архитектура позволяет легко добавлять новые источники данных.

## 🛠️ Технологический стек
- **Язык**: Python 3.12+
- **Логика**: Функциональное программирование + ООП (SOLID).
- **Тестирование**: Pytest (планируемое покрытие > 80%).
- **Инфраструктура**: Docker, Git.
- **Интерфейс**: Django (в разработке).

## 📂 Структура проекта
- `src/`: Ядро системы (двигатель, фильтры, утилиты).
- `main.py`: Точка входа в приложение.
- `tests/`: Набор автоматизированных тестов.
